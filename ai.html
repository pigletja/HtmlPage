What is Artificial Intelligence?

Machine Learning

• 컴퓨터 스스로가 새로운 규칙을 발견할 수 있도록 인간처럼 학습할 수 있도록 하는 기술
• 머신러닝은 알고리즘을 이용해 데이터를 분석하고 분석을 통해 학습하며 학습한 내용을 바탕으로 판단이나 예측을 함
• 기계 학습은 스스로 학습하고 데이터를 처리하는 과정
• 빅 데이터 삽입 - 데이터를 분석하여 모형 생성 - 모델을 사용하여 의사 결정 예측

Deep Learning

• ANN(인공신경망) 상호 연결된 뉴런의 네트워크
• 다중 은닉층을 갖는 인공 신경망을 이용한 기계 학습 수행 기술
• "딥"은 연속적인 신경망의 깊은 레이어를 의미하고 신경망이 심화됨에 따라 성능이 향상됨

BigData

• 디지털 환경에서 수집된 데이터로 아날로그 환경에서 생산된 데이터에 비해 텍스트 영상 수치 데이터까지 포함한 방대한 데이터
• 빅데이터는 병렬 처리를 위한 컴퓨터 시스템을 필요로 할 만큼 충분히 큼
• 빅데이터는 비즈니스 또는 연구를 위한 가치 창출
• 빅데이터가 창출하는 가치를 보장하기 위해서는 타당성과 신뢰성이 확보되어야 함

AI 역사

❖ 1차 태동기(1950~1956) : Turing Test
• 기계가 인간만큼 지능적으로 행동할 수 있는지 여부를 결정하는 테스트
• 캡차(CAPTCHA)
• 2000년 미국 카네기멜론대 연구진이 인간과 로봇을 구분하기 위해 만든 튜링 테스트
• 로봇이 어려운 캐릭터를 제시하고 매칭이 되면 인간으로 인식하는 방식
❖ 2차 태동기(1956~1974) : 데이터 기반 분석 프레임워크 구성
❖ 1차 암흑기(1974~1980) : AI 역사의 첫 겨울
• 예상과 달리 낮은 연구 성과로 인해 인공지능에 대한 투자가 줄어들면서 재정 위기에 직면
• 낙관론의 거품이 걷히면서 인공지능은 비판의 대상이 됐고,
❖ 성장기(1980~1987) : Expert systems
• 인간이 특정 분야에 대해 가지고 있는 전문지식을 정리하고 표현하여 컴퓨터에 저장함으로써 일반인이 전문지식을 활용할 수 있도록 하는 시스템
❖ 2차 암흑기(1987~1993)
• 전문가 시스템을 유지하는 데 드는 비용이 높았고 전문가 지식을 추출하는 데 병목 현상이 발생하여 시스템을 유지하는 데 어려움
❖ 1차 성숙기(1993~2000년) : Neural Network
• 1990년대의 AI 연구는 인터넷과 함께 다시 한번 증가
• 규칙 기반 시스템에서 인텔리전스 시스템으로
• 인간의 뇌신경망을 모방한 인공신경망 연구는 인공지능 발전에 큰 영향
• 신경세포로 구성된 인간의 뇌는 시냅스를 통해 전기적 자극을 전달
• 이를 시뮬레이션한 인공신경망은 뉴런이 특정 자극을 받으면 신호가 다음 뉴런으로 전달되는 방식으로 작동
❖ 2차 성숙기(2000~2010년) : ML and DL
• Machine Learning
• 데이터는 알고리즘을 사용하여 분석되고, 분석을 통해 학습되며, 배운 것에 근거하여 판단되거나 예측됨
• Deep Learning
• 인간의 뇌 속 뉴런과 유사한 입력·숨김·출력층을 이용해 데이터를 학습하는 인공신경망에서 개발된 인공지능
❖ 3차 성숙기(2010~present) : AlphaGo
• 이세돌과 대결한 알파고리는 나날이 성장하며 알파고 제로
• 알파고 제로는 학습을 위해 기본적인 인간 데이터가 필요하지 않음
